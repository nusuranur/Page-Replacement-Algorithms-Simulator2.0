WHAT IS PAGE REPLACEMENT?

Page Replacement helps OS decide which memory page to replace when a new page needs to be loaded into RAM.

Learn More
FIFO

First-In-First-Out replaces the oldest loaded page in memory.

LRU

Least Recently Used replaces the page not used for the longest time.

OPTIMAL

Replaces the page that will not be used for the longest time in the future.

LIFO

Last-In-First-Out removes the most recently loaded page.
